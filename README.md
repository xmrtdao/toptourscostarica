## 🌴 About

**Top Tours Costa Rica** showcases the top 10 ranked tours in Costa Rica, as voted by TikTok travelers. Our mission is to help visitors discover the best Pura Vida adventures through authentic social proof and TikTok video content.

### ✨ Features

- 🏆 **Top 10 Ranked Tours** - Curated by TikTok community

- 📱 **Mobile-First Design** - Optimized for TikTok generation

## 🚀 Quick Start

### Local Development

```bash
# Clone the repository
git clone https://github.com/xmrtdao/toptourscostarica.git
cd toptourscostarica

# Open index.html in your browser
open index.html
# or
python3 -m http.server 8000
```

### Deploy to GitHub Pages

```bash
# Push to GitHub
git add .
git commit -m "feat: Initial release - Top 10 Costa Rica tours"
git push origin main

# Enable GitHub Pages:
# Settings → Pages → Source: main branch → Save
```
